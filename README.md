# AdvancedSearch
Advanced web search using keyword generator, selenium and web crawler

This App has 3 components:

Client - Just a react client.

Node Api - A node api that is responsible for using selenium to scrape links out of a search engine

Go Api - Responsible for creating keywords and crawling over the links generated 
from the Node Api to match the content of the page with the regex filter


How to start the app:
Just navigate into the client folder and run npm start
it will run both the api's and the client with just this command.
